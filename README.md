# 🚀 JS Galaxy Playground

An interactive frontend showcase website built with HTML, CSS, JavaScript, and Three.js. This project demonstrates modern frontend development through immersive 3D graphics, smooth animations, glassmorphism UI, responsive design, and interactive JavaScript applications.

---

## 🌟 Features

### 🎨 Modern UI Design
- Glassmorphism user interface
- Animated gradient effects
- Responsive layout
- Smooth scrolling
- Mouse glow effect
- Loading screen animation

### 🌌 3D Experience
- Three.js powered 3D scene
- Interactive galaxy background
- Animated stars and visual effects
- Dynamic lighting

### 💻 Skills Showcase
- Interactive skill cards
- Orbit skill system
- Frontend technology visualization

### ⚡ JavaScript Playground
- Calculator
- To-Do App
- Password Generator
- Form Validation
- Typing Animation
- Animated Counters

### 📊 Developer Statistics
- Animated number counters
- Project showcase section
- GitHub integration

### 📱 Responsive Design
- Mobile-friendly interface
- Tablet support
- Desktop optimized layout

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- WebGL
